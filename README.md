# Title: Koala Vacation Planner

## Purpose
Android mobile application that allows users to add, update, and delete vacations as well as 
excursions. Also allows for sharing vacations with others and setting notifications. 
Additional features include search and report  generation.

## Directions
Launch Vacation Planner app. On Home screen, tap "My Vacations" button to see a list of vacations. 
The My Vacations screen displays a list of vacations along with a button to add vacations. 
Tap on the vacation to update the vacation or "ADD" button to add a vacation. This will bring up 
Vacation Details screen. In Vacation Details screen, tap 3 dots on top menu to delete/Share/Set 
Alerts on that vacation. Top section displays the vacation details where user can update vacation 
details. Bottom section displays a list of associated excursions. Tap on the excursion to update 
that excursion or tap the floating action button to add an excursion. This will bring up Excursion 
Details screen. This screen displays excursion details where user can update details or add 
excursions. Tap 3 dots on top menu to delete/Set Alerts on that excursion. Bottom section shows a 
spinner to change that excursion to a different vacation.
